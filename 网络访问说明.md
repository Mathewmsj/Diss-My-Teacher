# 网络访问说明

## 访问地址
- **前端地址**：`http://10.0.163.8:8080`
- **后端 API**：`http://10.0.163.8:8081/api`

## 如果无法访问，请检查：

### 1. 确认在同一 WiFi 网络
确保你的设备和服务器在同一 WiFi 网络下。

### 2. 检查防火墙设置
- **Windows**：检查 Windows Defender 防火墙是否阻止了端口 8080/8081
- **Mac**：系统设置 → 网络 → 防火墙 → 检查是否阻止了连接
- **手机/平板**：通常不需要特殊设置

### 3. 测试连接
在浏览器中直接访问：
- `http://10.0.163.8:8080` （应该能看到登录页面）
- `http://10.0.163.8:8081/api/teachers/` （需要登录 token，但可以测试端口是否开放）

### 4. 如果还是无法访问
1. 确认服务器正在运行（询问管理员）
2. 尝试使用服务器的其他 IP 地址
3. 检查路由器是否阻止了这些端口
4. 尝试使用手机热点测试（排除路由器限制）

### 5. 常见错误
- **ERR_CONNECTION_REFUSED**：服务器未启动或端口未开放
- **ERR_TIMED_OUT**：防火墙阻止或网络不通
- **ERR_NETWORK_CHANGED**：网络切换，需要重新连接

## 端口说明
- **8080**：前端 Web 服务（HTTP）
- **8081**：后端 API 服务（HTTP）

这些端口通常不会被防火墙阻止，如果仍有问题，请联系管理员。
