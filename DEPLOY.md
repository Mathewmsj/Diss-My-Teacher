# 部署说明

## 启动脚本使用说明

### 快速开始

1. **直接IP访问（端口范围 5000-5010）**
   ```bash
   ./start.sh 5001 8080
   ```
   - 后端端口: 5001
   - 前端端口: 8080

2. **域名访问（mathew 的端口是 8806）**
   ```bash
   ./start.sh 8806 8807
   ```
   - 后端端口: 8806（或根据需要调整）
   - 前端端口: 8807（或根据需要调整）

### 停止服务

```bash
./stop.sh
```

### 查看日志

```bash
tail -f backend.log    # 后端日志
tail -f frontend.log   # 前端日志
```

### 访问地址

- **后端API**: `http://110.40.153.38:端口号/api`
- **前端**: `http://110.40.153.38:端口号`

### 域名访问（mathew）

- **HTTP**: `http://mathew.yunguhs.com`
- **HTTPS**: `https://mathew.yunguhs.com`
- 需要将应用运行在端口 8806 上

### 重要提示

1. **端口范围**：直接IP访问必须使用 5000-5010 之间的端口
2. **监听地址**：脚本已配置为监听 `0.0.0.0`，可以从外部访问
3. **虚拟环境**：脚本会自动激活 Python 虚拟环境
4. **后台运行**：使用 `nohup` 和 `&` 在后台运行，即使退出 SSH 也不会停止
